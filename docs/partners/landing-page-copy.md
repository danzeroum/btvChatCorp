## Secao: Como funciona

1. **Você se torna parceiro** — cadastro em 5 minutos via API ou portal
2. **Cria um workspace** para o seu cliente — subdominio, logo, cores
3. **Cliente faz upload dos documentos** — PDFs, DOCX, TXT
4. **RAG ativo em < 5 minutos** — o chatbot ja responde com base nos documentos
5. **Fine-tuning continuo** — cada feedback melhora o modelo automaticamente
